# DuckyWorld - Game Design Document

## 게임 컨셉

| 항목 | 내용 |
|------|------|
| 게임명 | DuckyWorld |
| 장르 | 캐주얼 미니게임 컬렉션 |
| 플랫폼 | iOS / Android / PC (전 플랫폼 오픈) |
| 화면 방향 | 세로 (모바일 기준) |
| 타겟 유저 | 전 연령대, 캐주얼 게이머 |
| 레퍼런스 | 길건너친구들 (비주얼), 미니게임천국 (게임 스타일) |
| 플레이 목표 시간 | 1회 1분 내외 |

## 캐릭터

- **메인 캐릭터**: 오리 (Ducky)
  - 보유 중인 로우폴리 3D 오리 모델링 활용
  - 각 미니게임마다 테마에 맞는 의상/상황으로 등장
  - 캐릭터마다 고유 컬러 보유 → UI 테마 색상에 반영

---

## UI 흐름 (UI Flow)

### 로비 화면
- **배경**: 들판에 보유한 오리 캐릭터들이 자유롭게 뛰어노는 3D 씬
  - 선택된 메인 캐릭터는 카메라 포커스 대상
  - SNG 요소는 추후 검토, 현재는 캐릭터 표시 용도만
- **좌측 상단 HUD**
  - 메인 캐릭터 프로필 이미지
  - 닉네임
  - 계정 레벨
  - 보유 골드
- **우측 하단**: 게임 선택 버튼

### 게임 선택 창
- 게임 선택 버튼 클릭 시 팝업 형태로 표시
- **레이아웃**: 좌 3 : 우 7 비율
  - **좌측 패널**: 미니게임 리스트
    - 기본 선택: 최근 플레이한 게임
  - **우측 패널**
    - 선택된 게임의 실시간 미리보기 (별도 카메라로 실제 플레이 화면 렌더링)
    - 미리보기 위에 터치 조작 손가락 힌트 오버레이
    - 게임 이름 / 간단 설명
    - 조작 방법 명시
    - 내 랭킹 순위
  - **하단**: 게임 시작 버튼

### 게임 씬
- 게임 선택 → **씬 전환 연출** → 게임 씬 로드
- **카운트다운**: 3, 2, 1 → 게임 시작
  - 카운트다운 중 조작 방법에 맞는 손가락 모양 힌트 표시
- **인게임 상단 HUD** (한 줄, 게임 화면 가리지 않게)
  - 내 프로필 / 닉네임
  - 보유 골드 (게임 중 골드 수집 실시간 반영)
  - 게임별 스코어 정보 (버틴 시간, 점수 등)

### 게임 오버 화면
- 결과 정보 표시 (스코어, 최고기록 갱신 여부 등)
- **이어하기 버튼** (최초 1회만 표시)
  - 광고 시청 → 이어하기
  - 골드 소비 → 이어하기 (검토)
  - 광고 제거 구매 유저 → 광고 없이 이어하기
  - 이어하기 사용 후 재게임오버 시 이어하기 버튼 미표시
- **나가기 버튼** → 로비로 복귀

## 공통 시스템

### 계정
- **게스트 로그인**: 빠른 시작 지원
- **소셜 로그인**: 구글, 애플 등 주요 플랫폼 연동
- 게스트 → 소셜 계정 연동 전환 가능 (데이터 유지)

### 재화
- **골드**: 게임 중 맵에서 수집, 이어하기 비용 등에 사용
  - 추가 사용처는 추후 결정

### 싱글플레이
- 게임별 랭킹 순위표 제공
- **이어하기**: 게임 오버 시 1회 한정
  - 광고 시청 또는 골드 소비로 이어하기
  - 광고 제거 구매 유저는 광고 없이 이어하기

### PvP
- **1:1 실시간 대전** (우선 구현 목표)
- 4인 대전은 모바일 화면 제약으로 추후 검토
- **능력치 / 장비 적용**: PvP에서도 캐릭터 고유 스탯 및 장비 옵션 그대로 적용
  - 티어 매칭으로 비슷한 육성 수준끼리 대전 → 능력치 격차 자연 완화
- **매칭 방식**
  - 랜덤 매칭 (티어 기반)
  - 방 생성 후 대기
  - 친구 초대
- **구현 방식**: 입력 동기화 기반 실시간 대전
  - 추천 솔루션: **Photon PUN2** (방 생성 / 랜덤 매칭 / 친구 초대 모두 지원, 무료 플랜 동시접속 20명)
- **화면 분할 방식** (게임 진행 방향에 따라 다름)
  - 세로 진행 게임 (아래→위, 달리기/좁은길 등): **좌/우 분할** (각자 절반 화면)
  - 가로 진행 게임 (좌→우, 스키/플래피덕/허들/슬라이드런 등): **상/하 분할** (각자 절반 화면)
  - 공유 화면 게임 (악당피하기 등): **분할 없이 같은 화면** 에서 두 캐릭터가 함께 플레이

### 티어 시스템
- **게임별 개별 티어** 운영 (게임 통합 티어 없음)
- PvP 승패에 따라 티어 포인트 증감
- 티어 구성: 추후 결정 (예: 브론즈 → 실버 → 골드 → 플래티넘 등)
- 같은 티어 내 유저끼리 우선 매칭
- 로비 메인 화면에 보유 게임 중 **가장 높은 티어 1개 표시** (자부심 요소)
- 시즌 종료 시 티어별 보상 + 탑 랭커 고유 보상 지급

### 친구 시스템
- 인게임 친구 목록 보유
- 친구 초대 PvP 대전 지원
- 친구 데이터는 Firebase에서 관리, Photon으로 초대 연동

### 싱글 랭킹 / 시즌
- 게임별 개인 랭킹 순위 제공 (능력치 / 장비 적용 상태 기록)
- 랭킹 갱신 주기: 5~10분 단위
- **시즌제** 운영
  - 시즌 종료 시 탑 랭커에게 유니크 칭호 또는 고유 보상 지급

### 튜토리얼
- 별도 튜토리얼 없음
- 게임 선택 창의 실시간 미리보기 + 손가락 힌트로 대체
- 카운트다운 중 조작 힌트 표시
- 특정 조건(예: 계정 레벨 5 달성) 시 PvP 등 신규 기능 튜토리얼 추가 예정

## 아트 방향

| 항목 | 내용 |
|------|------|
| 렌더 파이프라인 | URP |
| 3D 스타일 | 로우폴리, 아기자기한 색감 |
| UI 스타일 | 단색 심플 UI, 흰색 기반 + 캐릭터/게임 고유 컬러 적용 |
| UI 레퍼런스 | 길건너친구들 스타일 단순 사각형 버튼 |

### UI 컬러 테마
- **로비**: 선택된 메인 캐릭터의 고유 컬러를 전체 UI에 적용
  - 예) 풀 속성 오리 → 진한 초록 + 연두 메인 컬러
- **게임 선택창 / 인게임**: 선택된 게임의 고유 컬러 적용
