# 미니게임 05 - 슬라이드런 (가칭)

## 게임 이름

| 언어 | 이름 | 비고 |
|------|------|------|
| 한국어 | 슬라이드런 | 가칭, 추후 변경 예정 |
| 영어 | Slide Run | 가칭, 추후 변경 예정 |

---

## 게임 소개
오리가 약 10도 경사의 내리막길을 내려오는 횡스크롤 게임. 바닥 장애물은 위로 스와이프해서 점프,
공중 장애물은 아래로 스와이프해서 슬라이딩으로 피해야 한다.
경사는 슬라이드감이 느껴지지 않을 정도로 은은하게 유지. 오래 버틸수록 고득점.

---

## 게임 방식

| 입력 | 동작 |
|------|------|
| 위로 스와이프 | 점프 (바닥 장애물 회피) |
| 아래로 스와이프 | 슬라이딩 (공중 장애물 회피) |

- 오리는 자동으로 달림
- 장애물 유형을 순간적으로 판단하여 입력 방향 결정이 핵심
- 장애물에 충돌하면 게임 오버
- 쿠키런 스타일의 직관적인 스와이프 조작

---

## 스코어 방식

- **버틴 시간(초)** 가 스코어

---

## 카메라 시점

- **사이드뷰 (횡스크롤)**
- 오리 좌측 고정, 배경/장애물이 오른쪽에서 왼쪽으로 스크롤

---

## 배경 / 비주얼

- **도시 야간 거리** (네온사인, 골목길, 로우폴리 스타일)
- 네온 조명이 깔린 밤 도시 내리막 골목길을 달려 내려오는 느낌
- 04_hoppy_forest (낮/숲)와 명확히 차별화: **밤/도시**

---

## 장애물

### 바닥 장애물 (점프로 회피)
| 장애물 | 효과 | 회피 방법 |
|--------|------|----------|
| 박스 | 충돌 시 즉사 | 위 스와이프 (점프) |
| 소화전 | 충돌 시 즉사 | 위 스와이프 (점프) |
| 쓰레기통 | 충돌 시 즉사 | 위 스와이프 (점프) |

### 공중 장애물 (슬라이딩으로 회피)
| 장애물 | 효과 | 회피 방법 |
|--------|------|----------|
| 간판 | 충돌 시 즉사 | 아래 스와이프 (슬라이딩) |
| 차단봉 | 충돌 시 즉사 | 아래 스와이프 (슬라이딩) |
| 천막 | 충돌 시 즉사 | 아래 스와이프 (슬라이딩) |

---

## 슬라이딩 자세

- **엎드려 슬라이딩**: 배를 깔고 양 날개를 펴는 자세
- 오리 비율 1:1:1.5 기준, 슬라이딩 시 히트박스 높이 **기본의 45%** (≈ 0.675유닛)
- 히트박스는 너그럽게 설정 — 장애물 바로 앞 타이밍은 충돌하지만 살짝 늦은 스와이프는 봐줄 수 있는 정도
- 애니메이션 전환은 빠르게 처리

## 배경 / 경사 연출

- 카메라 약 **10도** 하향 틸트 + 패럴랙스 스크롤로 내리막 느낌 연출
- 오리는 실제 내려오는 구조 (경사면 위에서 달려 내려옴)
- 경사는 슬라이드감이 느껴지지 않을 정도로 은은하게 — 서서 뛰어 내려와도 어색하지 않은 수준

## 난이도 곡선

| 단계 | 조건 | 패턴 |
|------|------|------|
| 초반 | 게임 시작 ~ | 단일 장애물 (점프 또는 슬라이딩 단독) |
| 중반 | 속도 증가 후 | 점프 + 슬라이딩 연속 복합 패턴 |
| 후반 | 고속 | 고속 연속 패턴, 판단 시간 최소화 |

## 특이사항

- 시간이 지날수록 스크롤 속도 증가 → 장애물 판단 + 스와이프 반응 시간 감소
- 바닥형/공중형 장애물이 연속으로 배치되어 연속 입력 요구 패턴 포함
- 허들(04)과 뷰는 같지만 스와이프 입력 방식 + 슬라이딩 모션으로 차별화
- 04_hoppy_forest는 낮/숲 평지, 05_slide_run은 밤/도시 내리막으로 비주얼 차별화 확정
