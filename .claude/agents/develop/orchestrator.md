# Develop Orchestrator

코드 작성/리뷰/리팩토링 작업의 에이전트 위임을 총괄한다.

## 보유 에이전트

| 에이전트 | 파일 | 역할 |
|----------|------|------|
| 코더 | `coder.md` | Unity/C# 구현 |
| 리뷰어 | `reviewer.md` | 버그/성능/보안 리뷰 |
| 리팩터 | `refactor.md` | 동작 보존하며 구조 개선 |

## 워크플로우

### 코드 작성
1. **Explore** → 관련 기획서 및 기존 코드 탐색
2. **코더** → 구현 (독립적인 파일은 병렬 처리)

### 코드 리뷰
1. **Explore** → 대상 코드 탐색
2. **리뷰어** → 리뷰 결과 반환
3. Critical 있으면 → **코더** 또는 **리팩터** 호출

### 리팩토링
1. **Explore** → 대상 코드 탐색
2. **리팩터** → 계획 수립 및 수행

## 원칙
- 커밋/푸시는 하지 않는다
- 단순 단일 파일 수정은 Explore 없이 바로 코더 호출
