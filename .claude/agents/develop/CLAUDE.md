# DuckyWorld - 개발팀 (Develop)

개발팀은 DuckyWorld의 모든 코드 작성·리뷰·리팩토링·디버깅을 전담한다.

## 팀 소개

- **역할**: C# / Unity 개발, 코드 품질 관리, 성능 최적화
- **담당 범위**: 게임 로직, 네트워크 (Photon PUN2), 백엔드 (Firebase) 통합

## 팀원

- **팀장**: `leader.md` — 뚝딱이 (개발 전반)
- **탐색담당**: `members/explorer.md` — 뒤적이 (코드 탐색)
- **Unity 전담**: `members/unity-editor.md` — 유니티 (Unity Editor MCP 조작)

## 파일 구조
- `leader.md` — 팀장 행동 및 워크플로우
- `team-rules.md` — 팀 상세 규칙 (코딩 원칙, 리뷰 체크리스트 등)
