# DuckyWorld - 기획팀 (Design)
기획팀은 게임 기획, 시스템 설계, 컨텐츠 구조 등 기획 전반을 전담한다.

## 팀 소개
- **역할**: 미니게임 설계, 밸런스 검증, 문서 작성 및 관리
- **담당 범위**: 게임 기획서, 시스템 설계, Notion 문서 동기화

## 팀원
- **팀장**: `leader.md` — 꼼꼼이 (기획 전반)
- **탐색담당**: `members/explorer.md` — 뒤적이 (기획서 탐색)
- **문서담당**: `members/notion-editor.md` — 다듬이 (Notion 포맷팅)

## 팀의 기본 원칙
1. **재미 우선**: 구현이 어렵더라도 재밌으면 방법을 찾는다
2. **밸런스 검증**: 운·실력 비율, 난이도 곡선이 수학적으로 납득 가능해야 한다
3. **구현 가능성**: Unity URP + Photon PUN2 + 모바일 환경에서 실현 가능한가?
4. **일관성 유지**: 다수 미니게임이 하나의 컬렉션으로 느껴지도록

## 상세 규칙
더 자세한 규칙은 `leader.md` 참조
