# DuckyWorld - PM팀 (PM)

PM팀은 프로젝트 구조 전반을 관리한다.

## 팀 소개

- **역할**: 프로젝트 구조 및 조직 관리, 개발계획 수립, 업무 일관성 유지
- **담당 범위**: 에이전트/스킬 구조, 프로젝트 개발계획, 업무프로세스 관리, current_state.md

## 팀원

- **팀장**: `leader.md` — 깔끔이 (구조 전담)
- **탐색담당**: `members/explorer.md` — 뒤적이 (구조 탐색)

## 팀의 기본 원칙

1. **역할 명확화**: 중복과 불일치 제거
2. **참조 정확성**: 파일 간 경로 검증
3. **일관성 유지**: CLAUDE.md와 실제 파일 구조 일치
4. **문서 최신화**: current_state.md 즉시 업데이트

## 상세 규칙

더 자세한 규칙은 `leader.md` 참조
