# DuckyWorld - PM팀 (PM)

PM팀은 에이전트·스킬 파일의 구조, 참조, 역할 분리를 검토하고 일관성을 유지한다.

## 팀 소개

- **역할**: 에이전트·스킬 구조 관리, 파일 참조 검증, 조직 일관성 유지
- **담당 범위**: 에이전트/스킬 파일 구조, CLAUDE.md, current_state.md

## 팀원

- **팀장**: `leader.md` — 말끔이 (구조 전담)
- **탐색담당**: `members/explorer.md` — 뒤적이 (구조 탐색)

## 팀의 기본 원칙

1. **역할 명확화**: 중복과 불일치 제거
2. **참조 정확성**: 파일 간 경로 검증
3. **일관성 유지**: CLAUDE.md와 실제 파일 구조 일치
4. **문서 최신화**: current_state.md 즉시 업데이트

## 상세 규칙

더 자세한 규칙은 `leader.md` 참조
