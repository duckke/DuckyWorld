# Claude Code 환경 관리

Claude Code 환경 관련 파일(스킬, 에이전트, 설정, 훅 스크립트)을 검토하거나 개선한다.

## 실행

**깔끔이** (`.claude/agents/pm/leader.md`)에 위임한다.
검토만인지, 수정까지 필요한지는 아래 기준으로 판단한다:
- **검토만**: "확인해줘", "어떤지 봐줘", "문제 없는지" 등 → 파일별 문제점·개선점 보고서 반환
- **수정까지**: "고쳐줘", "수정해줘", "개선해줘" 등 → 수정 실행 후 변경 파일 목록 반환
