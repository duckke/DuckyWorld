# 깔끔이 위임

PM 작업을 PM팀장 깔끔이에게 위임한다.

## 사전 구체화 단계 (비서 필수)

깔끔이에게 넘기기 전, 비서는 사용자의 의도·요구사항·원하는 결과 형식을 파악한다.

1. **명확하게 전달된 경우** → 바로 위임해도 된다.
2. **부족한 부분이 있는 경우** → 아래 항목 중 불명확한 것을 몇 가지 추가 질문해 구체화한 뒤 위임한다.
   - 목표: 무엇을 설계·수정·정비하는지
   - 기대 산출물: 어떤 파일·문서가 결과물인지
   - 제약: 건드리면 안 되는 것, 커밋 여부 등
3. 위임 시 "깔끔이(PM팀장)에게 위임합니다"라고 명시한다.

## 실행 방법

현재 대화 맥락에서 목표·기대 산출물·제약을 파악한 뒤,  
**Agent 툴을 `subagent_type: "general-purpose"`** 로 호출하고 아래 형식의 프롬프트를 전달한다.

```
/Users/duck/Documents/Work/DuckyWorld/.claude/agents/pm/leader.md 를 읽고 해당 워크플로우를 따를 것.

목표: [한 문장]
기대 산출물: [구체적 결과물]
제약: [있으면 명시, 없으면 생략]
```

결과는 재요약 없이 그대로 사용자에게 전달한다.
