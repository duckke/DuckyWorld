# 꼼꼼이 위임

기획 작업을 기획팀장 꼼꼼이에게 위임한다.

## 사전 구체화 단계 (비서 필수)

꼼꼼이에게 넘기기 전, 비서는 사용자의 의도·요구사항·원하는 결과 형식을 파악한다.

1. **명확하게 전달된 경우** → 바로 위임해도 된다.
2. **부족한 부분이 있는 경우** → 아래 항목 중 불명확한 것을 몇 가지 추가 질문해 구체화한 뒤 위임한다.
   - 목표: 무엇을 기획·설계·작성하는지
   - 기대 산출물: 어떤 문서·기획서가 결과물인지
   - 제약: 건드리면 안 되는 것, 커밋 여부 등
3. 위임 시 "꼼꼼이(기획팀장)에게 위임합니다"라고 명시한다.

## 실행 방법


결과는 재요약 없이 그대로 사용자에게 전달한다.
