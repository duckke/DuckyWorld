# 아트 방향, UI 스타일, 오디오

## 아트 방향

| 항목 | 내용 |
|------|------|
| 렌더 파이프라인 | URP |
| 3D 스타일 | 로우폴리, 아기자기한 색감 |
| UI 스타일 | 단색 심플 UI, 흰색 기반 + 캐릭터/게임 고유 컬러 적용 |
| UI 레퍼런스 | 길건너친구들 스타일 단순 사각형 버튼 |

### UI 컬러 테마
- **로비**: 선택된 메인 캐릭터의 고유 컬러를 전체 UI에 적용
  - 예) 풀 속성 오리 → 진한 초록 + 연두 메인 컬러
- **게임 선택창 / 인게임**: 선택된 게임의 고유 컬러 적용
