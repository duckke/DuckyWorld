# 미니게임 04 - 폴짝폴짝 포레스트

## 게임 이름

| 언어 | 이름 | 번역/표현 방향 |
|------|------|---------------|
| 🇰🇷 한국어 | **폴짝폴짝 포레스트** | 폴짝폴짝 = 귀엽고 경쾌한 점프 의태어. 숲속을 뛰어다니는 오리의 활기찬 느낌 |
| 🇺🇸 영어 | **Hoppy Forest** | Hoppy = 폴짝폴짝의 경쾌한 호핑 느낌. 귀엽고 간결하게 전달 |

> 추후 다국어 추가 시 경쾌하고 귀여운 점프 느낌 + 숲 배경을 살리는 방향으로 번역

---

## 게임 소개
오리가 숲 오솔길을 자동으로 달리며 장애물을 점프로 피하는 게임.
낮은 장애물은 한 번 탭, 높은 장애물은 더블탭으로 넘고, 구멍은 타이밍 맞춰 뛰어넘어야 한다.
두꺼운 통나무 위에는 올라타서 달릴 수 있다. 속도가 빨라질수록 장애물 간격도 넓어져 컨트롤 가능성을 유지한다.

---

## 핵심 재미

> **한 줄 요약**: 점점 빨라지는 숲길에서 장애물 높이를 순간 판단하고 점프/더블점프를 선택하는 타이밍 쾌감

- **긴장감 유형**: 반사신경 / 판단력 (장애물 높낮이 구분 → 탭 횟수 결정)
- **감정 곡선**: 여유 → 긴장 → 아슬아슬 → 폭사 → 한판만더

---

## 게임 방식

| 입력 | 동작 |
|------|------|
| 화면 탭 (1회) | 점프 |
| 화면 탭 (공중에서 1회 추가) | 더블점프 |

- 오리는 자동으로 달림 (이동 속도 조절 불가)
- 장애물 충돌 시 넘어져 게임 오버
- 구멍에 빠지면 게임 오버
- 통나무 위에 올라타면 달리기 지속, 끝에서 자연스럽게 착지

## 장애물

| 장애물 | 효과 | 회피 방법 | 비고 |
|--------|------|----------|------|
| 그루터기 | 충돌 시 즉사 | 탭 1회 (일반 점프) | 낮은 장애물 |
| 바위 | 충돌 시 즉사 | 탭 1회 (일반 점프) | 낮은 장애물 |
| 덤불 | 충돌 시 즉사 | 탭 1회 (일반 점프) | 낮은 장애물 |
| 통나무 울타리 | 충돌 시 즉사 | 탭 2회 (더블점프) | 높은 장애물 |
| 큰 버섯 | 충돌 시 즉사 | 탭 2회 (더블점프) | 높은 장애물 |
| 나뭇가지 | 충돌 시 즉사 | 탭 2회 (더블점프) | 높은 장애물 |
| 절벽 틈 | 낙하 즉사 | 탭으로 뛰어넘기 | 구멍 |
| 끊어진 나무다리 | 낙하 즉사 | 탭으로 뛰어넘기 | 구멍 |
| 통나무 플랫폼 | - | 올라타서 달리기 | 오리 높이 두께, 끝에서 자연 착지 |

- 낮은 장애물 / 높은 장애물 실루엣 차이를 명확하게 설계 (순간 판단 용이하게)

## 난이도 곡선

| 구간 | 시점 | 변화 내용 |
|------|------|----------|
| 초반 | 0~20초 | 낮은 장애물 위주, 느린 속도, 넓은 간격 |
| 중반 | 20~40초 | 높은 장애물·구멍 등장, 속도 증가, 통나무 플랫폼 조합 |
| 후반 | 40초~ | 낮은/높은/구멍/통나무 복합 배치, 고속, 판단 시간 최소화 |

- **속도 변화**: 점진적 증가
- **밀도 변화**: 속도가 빨라질수록 장애물 간격도 함께 넓어져 컨트롤 가능성 유지

## 코드 구조 (v4 기준)

| 항목 | 내용 |
|------|------|
| GameType enum | HoppyForest |
| 입력 타입 (IGameInput) | TapInput (탭 1회 → 점프, 공중 탭 1회 추가 → 더블점프) |
| 씬 | Games/HoppyForest.unity |
| 폴더 | MiniGame/HoppyForest/ |

### 게임 상태 전이 (MiniGameModule 관리)

```
None → Ready   : MiniGameModule.OnEnter()
Ready → Playing : UI 카운트다운 완료 → MiniGameModule.StartGame()
Playing → End    : 장애물 충돌 또는 구멍 낙하 → MiniGameModule.EndGame()
End → Ready      : UI 재시작 → MiniGameModule.RestartGame()
```

### MapManager 구현체 (HoppyForestMapManager)

- 공통: mapWidth, mapHeight, scrollSpeed, spawnArea, despawnArea
- **고유 정보**:
  - 장애물 종류별 배치 규칙 (낮은/높은/구멍/통나무 플랫폼)
  - 난이도 곡선 (속도 증가 + 장애물 간격 비례 확대)
  - 속도-간격 연동: 속도가 빨라질수록 장애물 간격도 함께 넓어져 컨트롤 가능성 유지

### 장애물 오브젝트

- 낮은 장애물 (그루터기/바위/덤불): ObstacleBase (Instant)
- 높은 장애물 (통나무 울타리/큰 버섯/나뭇가지): ObstacleBase (Instant)
- 구멍 (절벽 틈/끊어진 나무다리): 별도 처리 (낙하 즉사)
- 통나무 플랫폼: 올라타서 달리기 가능
