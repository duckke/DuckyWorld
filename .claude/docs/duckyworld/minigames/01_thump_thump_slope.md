# 미니게임 01 - 두근두근 슬로프

## 게임 이름

| 언어 | 이름 | 번역/표현 방향 |
|------|------|---------------|
| 🇰🇷 한국어 | **두근두근 슬로프** | 두근두근 = 심장 쫄깃한 긴장감. 아슬아슬하게 피하면서 내려오는 느낌을 직관적으로 전달 |
| 🇺🇸 영어 | **Thump Thump Slope** | 두근두근의 심장 소리를 의성어로 옮김. 리듬감 유지, 귀엽지만 스피드감 있는 뉘앙스 |

> 추후 다국어 추가 시 "두근두근"의 심장 쫄깃함 + 슬로프 스피드감 두 가지를 모두 살리는 방향으로 번역

---

## 게임 소개
오리가 스키를 타고 산을 내려가며 장애물을 피해 최대한 오래 버티는 게임.
타이밍에 맞게 터치하여 방향을 꺾으며 나무, 눈사람, 눈덩이 등의 장애물을 피해야 한다.
시간이 지날수록 속도가 빨라져 점점 더 세밀한 조작이 요구된다.

---

## 핵심 재미

> **한 줄 요약**: 점점 빨라지는 속도 속에서 터치 한 번의 타이밍으로 장애물을 아슬아슬하게 피하는 쫄깃한 긴장감

- **긴장감 유형**: 타이밍 / 반사신경
- **감정 곡선**: 여유 → 긴장 → 아슬아슬 → 폭사 → 한판만더

---

## 게임 방식

| 입력 | 동작 |
|------|------|
| 화면 터치 (1회) | 현재 진행 방향 기준으로 반대 방향으로 꺾기 |

- 초기 진행 방향: 왼쪽 대각선 아래
- 터치할 때마다 왼쪽 ↔ 오른쪽 대각선 아래로 번갈아 전환
- 즉, 터치 타이밍이 곧 장애물 회피의 핵심

## 장애물

### 즉사 장애물
| 장애물 | 효과 | 회피 방법 | 비고 |
|--------|------|----------|------|
| 🌲 나무 | 즉사 (게임 오버) | 터치로 방향 전환 | 고정 장애물 |

### 스턴 장애물
| 장애물 | 효과 | 회피 방법 | 비고 |
|--------|------|----------|------|
| ☃️ 눈사람 | 0.3초 스턴 (방향 전환 불가) | 터치로 방향 전환 | 고정 장애물, 머리 위 빙글빙글 스턴 이펙트 |
| ❄️ 눈덩이 | 0.3초 스턴 | 터치로 방향 전환 | 좌우에서 굴러오는 이동 장애물, 충격 스턴 이펙트 |

| 장애물 | 예정 효과 | 보류 이유 |
|--------|-----------|-----------|
| 🧊 얼음판 | 캐릭터가 바라보는 방향은 전환되지만 기존 진행 방향으로 계속 미끄러짐 (0.3초) | 메커니즘 구현 복잡도 및 플레이어 혼란 가능성 검토 필요, 추후 재논의 |

## 난이도 곡선

| 구간 | 시점 | 변화 내용 |
|------|------|----------|
| 초반 | 추후 논의 | 느린 속도, 장애물 간격 넓음, 적응 구간 |

## 코드 구조 (v4 기준)

| 항목 | 내용 |
|------|------|
| GameType enum | ThumpThumpSlope |
| 입력 타입 (IGameInput) | TapInput (화면 터치 1회 → 방향 전환) |
| 씬 | Games/ThumpThumpSlope.unity |
| 폴더 | MiniGame/ThumpThumpSlope/ |

### 게임 상태 전이 (MiniGameModule 관리)

```
None → Ready   : MiniGameModule.OnEnter()
Ready → Playing : UI 카운트다운 완료 → MiniGameModule.StartGame()
Playing → End    : 즉사 장애물(나무) 충돌 → MiniGameModule.EndGame()
End → Ready      : UI 재시작 → MiniGameModule.RestartGame()
```

### MapManager 구현체 (ThumpThumpSlopeMapManager)

- 공통: mapWidth, mapHeight, scrollSpeed, spawnArea, despawnArea
- **고유 정보**:
  - 장애물 배치 규칙 (나무/눈사람/눈덩이 밀도, 간격)
  - 난이도 곡선 (시간 → 스크롤 속도 증가 + 장애물 밀도 증가)
  - 절차적 생성 로직 (랜덤 배치)

### 장애물 오브젝트

- 나무: ObstacleBase (obstacleEffectType: Instant, hp=0)
- 눈사람: ObstacleBase (obstacleEffectType: Stun, effectValue: 0.3)
- 눈덩이: PassiveObject (자체이동, obstacleEffectType: Stun, effectValue: 0.3)

### 충돌 레이어

- 캐릭터: CollisionLayer.Character, collisionMask = Obstacle
- 장애물: CollisionLayer.Obstacle
