# 미니게임 05 - 슬라이드런 (가칭)

## 게임 이름

| 언어 | 이름 | 번역/표현 방향 |
|------|------|---------------|
| 🇰🇷 한국어 | **슬라이드런** | 가칭, 추후 변경 예정 |
| 🇺🇸 영어 | **Slide Run** | 가칭, 추후 변경 예정 |

> 추후 다국어 추가 시 내리막 슬라이딩의 스피드감을 살리는 방향으로 번역

---

## 게임 소개
오리가 약 10도 경사의 내리막길을 내려오는 횡스크롤 게임. 바닥 장애물은 위로 스와이프해서 점프,
공중 장애물은 아래로 스와이프해서 슬라이딩으로 피해야 한다.
경사는 슬라이드감이 느껴지지 않을 정도로 은은하게 유지. 오래 버틸수록 고득점.

---

## 핵심 재미

> **한 줄 요약**: 바닥/공중 장애물을 순간 판단하여 위/아래 스와이프를 선택하는 이분법적 반사신경 쾌감

- **긴장감 유형**: 반사신경 / 판단력 (장애물 위치 → 스와이프 방향 결정)
- **감정 곡선**: 단순 → 중독 → 아슬아슬 → 폭사 → 한판만더

---

## 게임 방식

| 입력 | 동작 |
|------|------|
| 위로 스와이프 | 점프 (바닥 장애물 회피) |
| 아래로 스와이프 | 슬라이딩 (공중 장애물 회피) |

- 오리는 자동으로 달림
- 장애물 유형을 순간적으로 판단하여 입력 방향 결정이 핵심
- 장애물에 충돌하면 게임 오버
- 쿠키런 스타일의 직관적인 스와이프 조작

## 슬라이딩 자세

- **엎드려 슬라이딩**: 배를 깔고 양 날개를 펴는 자세
- 오리 비율 1:1:1.5 기준, 슬라이딩 시 히트박스 높이 **기본의 45%** (≈ 0.675유닛)
- 히트박스는 너그럽게 설정 — 장애물 바로 앞 타이밍은 충돌하지만 살짝 늦은 스와이프는 봐줄 수 있는 정도
- 애니메이션 전환은 빠르게 처리

## 난이도 곡선

| 구간 | 시점 | 변화 내용 |
|------|------|----------|
| 초반 | 게임 시작~ | 단일 장애물 (점프 또는 슬라이딩 단독) |

- **밀도 변화**: 바닥/공중 장애물 연속 배치 패턴 증가, 연속 입력 요구

## 배경 / 경사 연출

- 카메라 약 **10도** 하향 틸트 + 패럴랙스 스크롤로 내리막 느낌 연출
- 오리는 실제 내려오는 구조 (경사면 위에서 달려 내려옴)
- 경사는 슬라이드감이 느껴지지 않을 정도로 은은하게 — 서서 뛰어 내려와도 어색하지 않은 수준

## 코드 구조 (v4 기준)

| 항목 | 내용 |
|------|------|
| GameType enum | SlideRun |
| 입력 타입 (IGameInput) | SwipeInput (위 스와이프 → 점프, 아래 스와이프 → 슬라이딩) |
| 씬 | Games/SlideRun.unity |
| 폴더 | MiniGame/SlideRun/ |

### 게임 상태 전이 (MiniGameModule 관리)

```
None → Ready   : MiniGameModule.OnEnter()
Ready → Playing : UI 카운트다운 완료 → MiniGameModule.StartGame()
Playing → End    : 장애물 충돌 → MiniGameModule.EndGame()
End → Ready      : UI 재시작 → MiniGameModule.RestartGame()
```

### 입력 처리

- InputManager에서 TouchDown 시 fingerId별 시작 position 저장
- TouchUp 시 delta 방향으로 스와이프 판정 (위/아래)
- 별도 SwipeInput 클래스는 입력 타입 선언 역할만 — 판정 로직은 InputManager에 집중

### MapManager 구현체 (SlideRunMapManager)

- 공통: mapWidth, mapHeight, scrollSpeed, spawnArea, despawnArea
- **고유 정보**:
  - 바닥/공중 장애물 배치 규칙
  - 연속 입력 요구 패턴 (점프 + 슬라이딩 연속)
  - 난이도 곡선 (스크롤 속도 점진적 증가 + 복합 패턴 빈도 증가)

### 장애물 오브젝트

- 바닥 장애물 (박스/소화전/쓰레기통): ObstacleBase (Instant)
- 공중 장애물 (간판/차단봉/천막): ObstacleBase (Instant)
