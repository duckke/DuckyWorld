# 미니게임 06 - 아슬아슬 좁은길 (가칭)

## 게임 이름

| 언어 | 이름 | 번역/표현 방향 |
|------|------|---------------|
| 🇰🇷 한국어 | **아슬아슬 좁은길** (가칭) | 아슬아슬 = 떨어질 것 같은 위태로운 긴장감. 좁은 길을 조심스럽게 통과하는 느낌을 직관적으로 전달 |
| 🇺🇸 영어 | **Narrow Path** (가칭) | 좁은 길이라는 직관적 의미. 심플하고 명확한 네이밍 |

> 추후 다국어 추가 시 "아슬아슬"의 위태로운 긴장감 + 좁은 길의 정밀 조작 느낌을 살리는 방향으로 번역

---

## 게임 소개
구불구불한 좁은 길을 가상 조이스틱으로 조작해 통과하는 게임.
빠를수록 고득점이지만 속도가 빠를수록 이탈 위험이 높아진다.
60초 타이머 안에 도착하면 남은 시간이 스코어.

---

## 핵심 재미

> **한 줄 요약**: 빠르게 가면 고득점이지만 떨어질 위험도 커지는, 속도와 안전 사이의 아슬아슬한 줄타기

- **긴장감 유형**: 정밀 조작 / 판단력
- **감정 곡선**: 여유 → 조심조심 → 점점 조급 → 이탈! → 한판만더

---

## 게임 방식

| 입력 | 동작 |
|------|------|
| 가상 조이스틱 | 오리 이동 방향 및 속도 조절 |

- 조이스틱 기울기에 따라 이동 속도 달라짐
- 길 바깥으로 이탈하면 게임 오버
- 섬세한 컨트롤이 핵심 (빠르게 가면 이탈 위험 ↑)

---

## 스코어 방식

| 항목 | 내용 |
|------|------|
| 기준 | 도착 시 남은 시간(초) — 60초 타이머에서 시작, 남은 시간이 높을수록 고득점 |
| 게임오버 조건 | 길 이탈 (낙하) 또는 60초 타이머 초과 |

---

## 카메라 시점

| 항목 | 내용 |
|------|------|
| 시점 | 쿼터뷰 (오리 등 + 머리가 살짝 보이는 각도) |
| 카메라 이동 | 캐릭터 추적 (오리를 화면 중앙에 고정) |
| 오리 위치 | 화면 중앙 고정 |

- 오리가 화면 아래에서 위 방향으로 진행

---

## 비주얼 / 분위기

| 항목 | 내용 |
|------|------|
| 배경 테마 | 절벽 산길 — 높은 산 중턱의 아찔한 외길 |
| 색감/톤 | 맑은 하늘 + 갈색/초록 산지, 따뜻한 오후 색감 |
| 바닥/지형 | 흙길 + 잔디, 양옆은 절벽/낭떠러지 |
| 주변 소품 | 이정표, 작은 돌, 야생화, 멀리 보이는 산봉우리 |
| 차별점 | 절벽 위 아찔한 높이감 — 낭떠러지 아래로 구름/안개가 보여 고소공포 연출 |

---

## 오리 애니메이션

| 상태 | 모션 설명 |
|------|----------|
| 기본 이동 | 뒤뚱뒤뚱 걷기 (조이스틱 강도에 따라 걷기 ↔ 달리기 전환) |
| 핵심 액션 | 달리기 시 날개를 양옆으로 벌리며 균형 잡기 |
| 피격/실패 | 길 이탈 시 날개를 파닥이며 아래로 추락 |
| 성공/클리어 | 도착 지점에서 만세 날갯짓 + 환호 |

## 장애물

| 장애물 | 효과 | 회피 방법 | 비고 |

## 난이도 곡선

| 구간 | 시점 | 변화 내용 |
|------|------|----------|
| 초반 | 0~20초 | 넓은 길, 완만한 커브 — 조작 적응 구간 |
| 중반 | 20~40초 | 길이 좁아지고 커브가 급해짐, 속도 조절 판단 필요 |
| 후반 | 40~60초 | 매우 좁은 길 + 급커브 연속, 정밀 조작 요구 |

- **속도 변화**: 플레이어 조이스틱 입력에 따라 결정 (빠르게 갈수록 고득점이지만 위험)
- **밀도 변화**: 구간이 진행될수록 길 폭이 좁아지고 커브 빈도·각도 증가
