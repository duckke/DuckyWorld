# 미니게임 07 - 피해피해 악당오리 (가칭)

## 게임 이름

| 언어 | 이름 | 번역/표현 방향 |
|------|------|---------------|
| 🇰🇷 한국어 | **피해피해 악당오리** (가칭) | 피해피해 = 급박하게 피하는 반복 느낌. 악당 오리를 필사적으로 피하는 긴장감 전달 |
| 🇺🇸 영어 | **Duck Dodge** (가칭) | Duck(오리) + Dodge(피하기)의 말장난. 짧고 직관적 |

> 추후 다국어 추가 시 "피해피해"의 급박한 반복 느낌 + 악당 오리라는 캐릭터성을 살리는 방향으로 번역

---

## 게임 소개
4x4 격자 공간에서 사방에서 돌진해오는 악당 오리를 가상 조이스틱으로 피하는 게임.
처음에는 한 마리씩 등장하지만 점점 두세 마리가 동시에 돌진해온다.
쉬워 보이지만 방향이 엇갈리며 생각보다 어렵다. 오래 버틸수록 고득점.

**PvP**: 화면 분할 없이 같은 격자 위에서 두 오리가 동시에 플레이.
같은 악당을 피하면서 상대가 먼저 맞으면 승리. 상대 위치를 보며 유리한 칸을 선점하는 심리전이 핵심.

---

## 핵심 재미

> **한 줄 요약**: 사방에서 돌진해오는 악당을 격자 위에서 아슬아슬하게 피하는 순간 판단의 쾌감

- **긴장감 유형**: 반사신경 / 판단력 / 심리전 (PvP)
- **감정 곡선**: 여유 → 긴장 → 혼란 → 폭사 → 한판만더

---

## 게임 방식

| 입력 | 동작 |
|------|------|
| 가상 조이스틱 | 4x4 격자 위에서 오리 이동 (격자 단위) |

- 이동은 격자 단위 (한 칸씩 이동)
- 악당 오리가 돌진해오는 방향을 보고 칸을 피해야 함
- 악당 오리에 충돌하면 게임 오버

## 난이도 곡선

| 구간 | 시점 | 변화 내용 |
|------|------|----------|
| 초반 | 0~15초 | 1마리씩 순차 등장, 돌진 속도 느림, 예고 시간 충분 |
| 중반 | 15~35초 | 2마리 동시 등장, 돌진 속도 증가, 방향 조합 다양화 |
| 후반 | 35초~ | 2~3마리 동시 등장, 고속 돌진, 예고 시간 최소화, 연속 공격 |

- **속도 변화**: 구간별 증가 (악당 오리 돌진 속도 + 등장 간격 단축)
- **밀도 변화**: 동시 등장 마릿수 증가 (1마리 → 2마리 → 2~3마리)

## 코드 구조 (v4 기준)

| 항목 | 내용 |
|------|------|
| GameType enum | DodgeVillain |
| 입력 타입 (IGameInput) | JoystickInput (가상 조이스틱 → 4x4 격자 칸 이동) |
| 씬 | Games/DodgeVillain.unity |
| 폴더 | MiniGame/DodgeVillain/ |

### 게임 상태 전이 (MiniGameModule 관리)

```
None → Ready   : MiniGameModule.OnEnter()
Ready → Playing : UI 카운트다운 완료 → MiniGameModule.StartGame()
Playing → End    : 악당 오리 충돌 → MiniGameModule.EndGame()
End → Ready      : UI 재시작 → MiniGameModule.RestartGame()
```

### 입력 처리

- InputData.joystickValue: 격자 단위 이동 방향 결정
- 4방향 이동 (위/아래/좌/우)

### MapManager 구현체 (DodgeVillainMapManager)

- 공통: mapWidth, mapHeight (4x4 격자 크기)
- **고유 정보**:
  - 악당 오리 등장 방향/타이밍/동시 마릿수 규칙
  - 난이도 곡선 (돌진 속도 + 등장 간격 단축 + 동시 등장 마릿수 증가)
  - 예고 모션 타이밍

### 장애물 오브젝트

- 악당 오리: PassiveObject (자체이동, 직선 돌진) 또는 별도 MonsterBase
  - 4방향(위/아래/좌/우)에서 직선 돌진

### 충돌 레이어

- 캐릭터: CollisionLayer.Character, collisionMask = Monster
- 악당 오리: CollisionLayer.Monster

## 특이사항

- 악당 오리 등장 방향: 위→아래 / 아래→위 / 좌→우 / 우→좌 번갈아 등장
- 초반: 1마리씩 순차 등장
- 시간 경과에 따라 2~3마리 동시 등장으로 난이도 증가
- 악당 오리 돌진 전 짧은 예고 모션(예: 발 구르기) 추가 검토 → 유저가 방향 예측 가능하게
- 마리오파티 미니게임 오마주
- 악당 캐릭터: 악당 오리 (별도 디자인 필요)
