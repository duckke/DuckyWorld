# DuckyWorld - Economy & Character System

## 재화

| 재화 | 획득 방법 | 주요 사용처 |
|------|----------|------------|
| 골드 | 게임 중 맵에서 수집, 일일 보상 등 | 뽑기, 스태미나 즉시 회복, 장비 강화 |

> 추가 재화(프리미엄 등)는 추후 검토

---

## 캐릭터 (오리) 시스템

### 획득
- **뽑기**: 알을 깨고 랜덤으로 오리 등장
  - 뽑기 비용: 골드 (구체적인 수치는 추후 결정)

### 스태미나
- 오리마다 개별 스태미나 보유
- 게임 플레이 시 스태미나 소모
- 스태미나 소진 시 해당 오리로 플레이 불가 (다른 오리로는 가능)
- 오리가 1마리뿐이고 스태미나 소진 → 게임 불가 → 뽑기 유도
- **회복 방식**
  - 시간 경과 → 자동 회복
  - 골드 소비 → 즉시 회복

### 캐릭터 고유 스탯
- 오리마다 기본 스탯 보유 (예: 스키 특화 오리는 기본 이동속도 낮음)
- 스탯은 장비로 추가 커스터마이징 가능

### 캐릭터 경험치 / 레벨
- 게임 플레이 시 경험치 획득
- 레벨업 시 새로운 게임 또는 컨텐츠 오픈

---

## 장비 시스템

### 장착 효과
- 외형 변화 (코스튬, 아이템 착용 비주얼)
- 고강화 달성 시 이펙트 추가 (빛남, 파티클 등)
- 능력치 스탯 부여

### 능력치 스탯 후보

| 스탯 | 스키 | 달리기 | 폭탄피하기 | 비고 |
|------|------|--------|-----------|------|
| 이동속도 ↑ | 불리 | 유리 | 유리 | 게임마다 유불리 다름 |
| 이동속도 ↓ | 유리 | 불리 | 불리 | |
| 골드 획득량 ↑ | 유리 | 유리 | 유리 | 전 게임 공통 |
| 스턴 저항 | 유리 (눈사람) | - | - | 게임별 의미 다름 |
| 시작 무적 시간 | 유리 | 유리 | 유리 | 전 게임 공통 |
| 스코어 보너스 % | 유리 | 유리 | 유리 | 랭킹에 영향 |

> 유저가 게임에 맞는 오리 + 장비 조합을 선택하는 전략 요소

### 강화
- 골드 소비로 장비 강화
- 강화 수치 증가 → 능력치 상승 + 특정 단계마다 외형/이펙트 변화

---

## PvP 능력치 적용 방침

- **PvP에서도 능력치 / 장비 옵션 그대로 적용**
- 티어 기반 매칭으로 비슷한 육성 수준끼리 대전 → 격차 자연 완화
- 싱글 랭킹도 능력치 적용 상태로 기록

---

## 골드 순환 구조

```
게임 플레이
    ↓
골드 획득 (맵 수집)
    ↓
   사용처
    ├─ 뽑기 (새 오리 획득)
    ├─ 스태미나 즉시 회복
    ├─ 장비 강화
    └─ (추후 추가 검토)
```
